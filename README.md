# League of Legends AI Analyst

Riot APIを利用してプレイヤーの直近の試合データを取得し、GoogleのGemini APIを使ってプロのコーチのように辛口かつ的確な分析を行うツールです。

## 特徴
- **ディープなデータ取得**: OP.GGなどの統計サイトのように、対面のランク格差、キル関与率（KP）、10分時点のCS、ダメージシェア、対物ダメージなどの詳細な「隠しスタッツ」をRiot APIから抽出します。
- **AIによる高度な分析**: Geminiが持つチャンピオンごとのマッチアップの知識を活用し、「有利な対面なのに勝てていない」「味方が弱すぎただけで自分は悪くない（しょうがない試合）」といった客観的な敗因特定やプロファイリングを行います。
- **柔軟な出力モード**: AI分析を行うだけでなく、抽出した生データをJSONやコンパクトなテキスト形式でダンプ（保存）して後から人間が見直すことも可能です。

## 前提条件
実行には以下のAPIキーが必要です。
1. **Riot API Key**: [Riot Developer Portal](https://developer.riotgames.com/) から取得
2. **Gemini API Key**: [Google AI Studio](https://aistudio.google.com/app/apikey) から取得

## セットアップ

1. リポジトリをクローンまたはダウンロードします。
2. 必要なライブラリをインストールします。
   ```bash
   pip install -r requirements.txt
   ```
3. プロジェクトのルートディレクトリに `.env` ファイルを作成し、以下の内容を記述します。
   ```env
   # .env ファイルの例
   RIOT_API_KEY="YOUR_RIOT_API_KEY"
   GEMINI_API_KEY="YOUR_GEMINI_API_KEY"
   
   # 使用するGeminiのモデル（任意）
   # GEMINI_MODEL="gemini-2.5-flash"
   ```

## 使い方

以下のコマンドでツールを起動します。

```bash
python main.py
```

### 実行の流れ
1. **Riot IDの入力**: 分析したいプレイヤー名とタグを `PlayerName#TAG` の形式で入力します。
2. **実行モードの選択**:
   - `1`: AI分析（デフォルト）。データを取得後、Gemini APIを使ってコーチング結果を出力します。
   - `2`: JSONダンプ。AI分析は行わず、取得したデータをJSONとテキストファイルとして保存します（API節約用）。
3. **試合数の入力**: 取得したい直近の試合数を入力します（デフォルトは10試合）。
4. **テーマの入力（任意）**: 「序盤のガンクへの対応を見てほしい」「ビルドの指摘をしてほしい」など、AIに重点的に分析してほしいテーマを入力できます。空欄でEnterを押すと全体的な分析を行います。

※モード1（AI分析）を選んだ場合でも、裏側で自動的に `dump_プレイヤー名.json` と `dump_プレイヤー名.txt` が保存されます。

## 出力ファイル
- `dump_プレイヤー名.json`: Riot APIから取得し、整理した全試合の生データです。
- `dump_プレイヤー名.txt`: AIに送信されるのと同じ形式の、人間にも読みやすいコンパクトな試合のサマリーテキストです。

## 注意事項
- Riotの無料APIキーは**24時間で有効期限が切れる**ため、毎日再発行して `.env` を更新する必要があります。
- 無料APIの制限（2分間に100リクエスト）を回避するため、対面相手のランクのみを取得するように最適化されていますが、一度に大量の試合数（20試合以上など）を指定するとエラーになる可能性があります。
